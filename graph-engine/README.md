# Graph Engine

ShadowGraph graph intelligence module for constructing and querying identity relationships in Neo4j.

## Responsibilities
- Normalize identity entities into graph nodes.
- Generate relationship edges from AI similarity scores.
- Persist and query risk-oriented graph topology.

## Node Types
- `identity`
- `account`
- `image`
- `post`
- `research_artifact`

## Edge Types
- `username_similarity`
- `image_similarity`
- `text_similarity`
- `network_relationship`
